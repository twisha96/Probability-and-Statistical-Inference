# Method of Moments

Method of Moments is a method of estimation of population parameters. Using the method of moments, the script estimates the population parameter for the following distributions :  
1. Bernoulli.  
2. Binomial.  
3. Geometric.   
4. Poisson.  
5. Uniform.  
6. Exponential.  
7. Normal.   
8. Gamma.  
9. Beta.  
10. Chi-square.
